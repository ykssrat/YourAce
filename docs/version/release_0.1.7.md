YourAce 0.1.7 界面更新说明

版本号：0.1.7
整理日期：2026-05-06
文档性质：已上线说明 + 界面更新摘要

一、本次界面更新了什么

- 看盘页新增了分钟图展示，直接把自选池的分钟量做成可视化条形图。
- 自选卡片里继续保留最新价、涨跌幅、成交量、绑定 ETF、板块名称和信号原因，信息密度比上一版更高。
- 登录页继续使用低成本的用户名/密码注册登录方案，手机号或微信号可以直接作为账号名使用。
- 策略入口继续收敛为已实现项，前端不再把占位策略伪装成可点击功能。

二、分钟图补了什么

- 当前分钟图基于后端返回的分钟成交量序列绘制，不依赖额外图表库。
- 图上展示最近若干分钟的柱状变化，并突出最新一根。
- 这个图是分钟级节奏可视化，不是逐笔 tick 图，也不是完整分时 K 线图。
- 如果后续要进一步增强，可以继续补分钟价格曲线、涨跌区间、成交量均线和缩放手势。

三、看盘页的用户感知变化

- 用户进入看盘页后，能直接看到自选池每只标的的分钟图、最新价格和当前动作建议。
- 最新提醒区保留了通知摘要，能快速看出哪些标的触发了买卖信号。
- 自动刷新仍然保留，同时支持手动立即刷新，适合弱网或临时校验场景。

四、版本边界

- 这次更新补的是界面表达，不是新策略逻辑。
- 分钟行情能力仍然是分钟级快照 + 推送，不是交易终端式逐笔实时盯盘。
- 如果用户要更像专业行情软件的效果，下一步应继续补分时价格曲线和更完整的盘中图层。

五、后续可继续做的事

1. 给分钟图补价格曲线，而不只是成交量柱。
2. 给卡片加盘中高低点标注，提升可读性。
3. 给看盘页补一层可折叠的分钟详情面板。
