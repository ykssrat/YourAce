YourAce 移动端说明

一、当前功能

- 搜索建议与代码检索
- 自选代码增删
- 评分仪表盘与三维信号展示
- 后端地址可配置（局域网）
- 分析请求超时、重试与多基地址回退
- 新闻系统开关（最小闭环验证可关闭）

二、关键交互流程

1. 输入或选择代码
2. 配置后端地址（例如 10.198.35.177:8000）
3. 选择是否开启新闻系统
4. 点击分析并展示结果

三、网络设置建议

- 真机调试优先使用电脑局域网 IP
- 模拟器可用 10.0.2.2:8000
- 127.0.0.1 在真机上指向手机本机，不是电脑

四、常见失败原因

- 后端进程未启动
- 电脑防火墙未放行 8000 端口
- 手机与电脑不在同一局域网
- 地址输入格式错误（建议带端口）
