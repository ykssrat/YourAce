YourAce 视觉主题说明

适用版本：0.1.6 起
一、当前启动页与整体 UI 的核心口号

- Slogan：您的投资，更有章法。

口号作为启动页文案使用。

二、主题色基线
当前建议固化以下三种基础主题色，作为 YourAce 的主视觉基调：

- 深蓝色：`#013E75`
- 深红色：`#a42423`
- 米白色：`#fffbf0`

三、颜色角色建议

1) 深蓝色 `#013E75`

- 用途：主品牌色、标题色、导航强调色、理性信息主色
- 气质：专业、稳定、克制、可信
- 建议场景：
  - 启动页主图标辅助色
  - 顶部标题
  - 主按钮或选中态
  - 结构化信息区块边框/图标

2) 深红色 `#a42423`

- 用途：品牌辅助强调色、关键视觉锚点、启动页标语强调
- 气质：力量、决断、市场感、金融气质
- 建议场景：
  - Slogan 文字
  - 启动页图标中的重点元素
  - 关键状态强调
  - 少量高对比 CTA 点缀

说明：
- 深红色应作为强调色控制使用，不建议在大面积正文区域铺开，否则容易产生压迫感。

3) 米白色 `#fffbf0`

- 用途：主背景色、卡片底色、留白承载色
- 气质：柔和、克制、减少金融类产品常见的强刺激感
- 建议场景：
  - 启动页背景
  - 卡片背景
  - 页面大面积底色
  - 内容留白区域

四、整体设计原则

品牌主题建议遵循以下方向：

- 强调“有章法”的感受：
  - 版面要整齐
  - 信息层级要清晰
  - 配色要克制
  - 强调色要少而准

五、推荐配色关系

- 页面主背景：`#fffbf0`
- 主标题 / 主信息：`#013E75`
- Slogan / 强调信息：`#a42423`
- 次级边框 / 分割区域：可在主色基础上降低透明度或使用更浅同系色
